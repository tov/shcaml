Module StringShtream

module StringShtream: sig .. end
Shtreams of strings. This module is the result of applying AnyShtream.Make to the module StringShtream.StringElem. Thus, shtreams handled by this module are compatible with the shtreams of Shtream, LineShtream, and modules created by AnyShtream.Make, but this module provides additional functions for reading and writing string Stream.ts.
module StringElem: AnyShtream.ELEM 
  with type 'a elem = string
   and type initial = unit
